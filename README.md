# Explorando a pasta `/home` no Linux
## Porque Começar na `/home`?
A pasta `/home` é comoa casa dos usuários no S.O. Linux. Cada pessoa que usa o sistema tem sua pasta aqui, com seus arquivos, fotos, etc.
